Asosiy modul, to'liq ekranni qoplagan butun sahifa bo'ladi.
Sahifada elementlarni tanlash orqali, ko'plab ma'lumotlarni olish imkoniyati bo'lishi kerak. 
Sahifa juda ham ajoyib, wow effektlar, ajoyib, kreativ va silliq animatsiyalar bo'lishi kerak.

Ushbu asosiy modul juda katta monitorda ko'rsatilinadi, shu sababli elementlar ko'proq joylashtirilishi kerak. 
Default font size 10px - 12px bo'ladi max. 

Xaritada hech qanday obekt belgilanmagan bo'lsa, sarnovul mahallasining umumiy ma'lumotlari ko'rinib turishi kerak. Yuqori top barda cardlar bo'ladi, cardlarda barcha modullar bo'yicha ma'lumotlar ko'rsatiladi. Aholi soni, Soliq, Ta'lim, Yoshlar, IIB, Gaz, Obodonlashtirish, Suv, Elektr, Yer kadastri bo'yicha ma'lumotlar bo'ladi. Top barda cardlar 2qatordan 8ta ustun bo'ladi. Umumiy 16ta card.

Bottom barda esa yana nimadirlar ko'rinib turishligi kerak. Element belgilanganida o'ng tomonning yarmida ko'plab statistika va element to'g'risida juda ham ko'plab ma'lumotlar ko'rsatilinishi kerak.
